________________________________________사용 예제__________________________________________________ 라이브러리 : 자주 사용하는 로직을 재사용할 수 있도록 고안된 소프트웨어 jQuery : jQuery는 DOM을 내부에 감추고 보다 쉽게 웹페이지를 조작할 수 있도록 돕는 도구. $()는 jQuery의 함수. 이 함수의 인자로 CSS선택자(li)를 전달하면 jQuery 객체라는 것을 리텀함. 이 객체는 선택자에 해당하는 element를 제어하는 다양한 메소드를 가지고 있음.
