문자열은 분명히 프로퍼티와 메소드가 있다. 그렇다면 객체이다. 그런데 왜 문자열이 객체가 아니라고 할까? 그것은 내부적으로 문자열이 원시 데이터 타입이고 문장열과 관련된 어떤 작업을 하려고 때 자바스크립트는 임시로 문자열 객체를 만들고 사용이 끝나면 제거하기 때문이다. 이러한 처리는 내부적으로 일어난다. 그렇기 떄문에 몰라도 된다. 하지만 원시 데이터 타입과 객체는 좀 다른 동작 방법을 가지고 있기 때문에 이들을 분별하는 것은 결국엔 필요하다. 원시 데이터 타입 종류 1. string 2. Number 3. Boolean 4. Null 5. undefined
